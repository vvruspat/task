## Customization

MGallery is styled with CSS variables. See **Guides/Theming & Customization** for the
override mechanism (global / zonal / instance).

| Token | Controls | Default |
|-------|----------|---------|
| `--gallery-gap` | Gap between header/slider/footer | `--gap-m` |
| `--gallery-radius` | Slider container corner radius | `--radius-lg` |
| `--gallery-inset` | Offset of nav buttons & tools from the edges | `--gap-m` |

Nav buttons are `MButton` (`mode="round"`) instances — see **MButton › Customization** to
restyle them.
