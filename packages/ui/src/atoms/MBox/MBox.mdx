# Definition

Box is a layout primitive whose job is to control spacing through props.

It exposes `padding`/`margin` shorthands plus per-side (`paddingTop`, `paddingRight`, `paddingBottom`, `paddingLeft`) and per-axis (`paddingX`, `paddingY`) overrides, and the same set for `margin`. More specific props win: a side prop overrides its axis prop, which overrides the shorthand.
