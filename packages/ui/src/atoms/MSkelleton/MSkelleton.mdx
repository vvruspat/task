## Customization

MSkelleton is styled with CSS variables. See **Guides/Theming & Customization** for the
override mechanism (global / zonal / instance).

| Token | Controls | Default |
|-------|----------|---------|
| `--skelleton-radius` | Corner radius | `--radius-md` |
| `--skelleton-min-size` | Minimum width (per `minSize`) | `calc(4 * --gap-{xs,s,m,l,xl,2xl,3xl,4xl})` |

Surface colors (`--color-surface-sunken`, `--color-surface-overlay`) are shared semantic
roles — see **Guides/Theming & Customization** to recolor them for a whole brand.
