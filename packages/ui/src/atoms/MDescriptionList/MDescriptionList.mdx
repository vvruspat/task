## Customization

MDescriptionList is styled with CSS variables. See **Guides/Theming & Customization** for
the override mechanism (global / zonal / instance).

| Token | Controls | Default |
|-------|----------|---------|
| `--descriptionlist-padding-x` | Horizontal padding of the list | `--gap-l` |
| `--descriptionlist-row-gap` | Gap between rows | `--gap-s` |
| `--descriptionlist-column-gap` | Gap between term and description columns | `--gap-l` |
| `--descriptionlist-item-py` | Vertical padding of each `dt`/`dd` | `--gap-xs` |

Term/description colors (`--color-text-secondary`, `--color-text-primary`) and the row
background (`--color-surface`) are shared semantic roles — see
**Guides/Theming & Customization** to recolor them for a whole brand.
