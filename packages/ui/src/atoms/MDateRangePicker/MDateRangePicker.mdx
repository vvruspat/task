## Customization

MDateRangePicker composes `MInput`, `MButton`, `MDropdown` and `MCalendar`, each already
themeable through their own `--input-*` / `--button-*` / `--dropdown-*` / `--calendar-*`
tokens — see **Guides/Theming & Customization**. It exposes one token of its own:

| Token | Controls | Default |
|-------|----------|---------|
| `--date-range-picker-gap` | Gap between the "from" input, separator, and "to" input | `--gap-xs` |
